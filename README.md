# vaibhav-00011.github.io

My personal portfolio website hosted using GitHub Pages.

## Project Structure

```
├── index.html       # Main portfolio page
├── style.css        # Glassmorphism styling and animations
├── script.js        # Interactive mouse parallax effect
└── resume.pdf       # Resume document
```

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/vaibhav-00011/vaibhav-00011.github.io.git
   ```
2. Open `index.html` in your browser to view the portfolio locally.

## Troubleshooting

### IDE Plugin Compatibility Errors

If you see an error like:

> Plugin 'GitHub Copilot' is not compatible with the current version of the IDE

This means your GitHub Copilot plugin version doesn't match your IDE version. To fix this:

1. **Update the plugin**: Go to **Settings → Plugins → Installed**, find GitHub Copilot, and click **Update**.
2. **Or install from the marketplace**: Go to **Settings → Plugins → Marketplace**, search for "GitHub Copilot", and install the latest compatible version.
3. **Restart your IDE** after updating.

> **Note:** You don't need an IDE to work on this project. You can edit the files with any text editor and preview `index.html` directly in a browser.
